# 视图

- 是什么？

  是一个虚拟存在的数据表，这个虚拟表并不在数据库中**实际**存在

- 作用是什么？

  将一些较为复杂查询语句的结果，封装到一个虚拟表中，后期再有相同需求时，直接查询该虚拟表即可
